Configuration

If you want to configure the registration of the MVC interceptor i.e. enable/disable it, then, in your Spring configuration files like application.properties for Spring Boot or other files you import with ways like:
                                        @PropertySource(value = {"classpath:configuration/MVCConfiguration.prop"})
then, if you want to enable this plugin then the configuration is: 
                                        orionlibs.spring_http_request_logger.interceptor.enabled=true
The configuration for the logger is as follows:
                                        orionlibs.spring_http_request_logger.log.ip.address.enabled=true
                                        orionlibs.spring_http_request_logger.log.http.method.enabled=true
                                        orionlibs.spring_http_request_logger.log.uri.enabled=true
                                        orionlibs.spring_http_request_logger.log.pattern.for.each.log.record.element=%s: %s
The pattern %s: %s makes the log look like:
Jul 07, 2023 10:57:44 AM io.github.orionlibs.spring_http_request_logger.LoggingInterceptor preHandle INFO: IP: 127.0.0.1, URI: GET /api/v1/users/1234
                                        orionlibs.spring_http_request_logger.log.http.methods.logged=* OR orionlibs.spring_http_request_logger.log.http.methods.logged=GET,POST,PUT,DELETE,etc.
                                    
                                        orionlibs.spring_http_request_logger.log.uris.logged.pattern=* (accepts all URIs) OR ^(/[-a-zA-Z0-9+&@#/%?=~_|!:,.;]*)$ (accepts all valid URIs)
                                        OR
                                        orionlibs.spring_http_request_logger.log.uris.logged.pattern=^(/api/v1/[-a-zA-Z0-9+&@#/%?=~_|!:,.;]*)$ (accepts all valid URIs that starts with /api/v1/)
                                        OR
                                        orionlibs.spring_http_request_logger.log.uris.logged.pattern= OR .*/users/.* (accepts all valid URIs  that contains /users/)
                                    
                                        orionlibs.spring_http_request_logger.log.uri.query.params.enabled=false
the above configuration is false by default, but if you make it true then if the HTTP request URI had request parameters then the log will look like:
INFO: IP: 127.0.0.1, URI: GET /api/v1/search?query=some+query&options=4
                                        orionlibs.spring_http_request_logger.log.request.processing.duration.enabled=false
the above configuration is false by default, but if you make it true then there will be a log that looks like:
INFO: Handler: com.github.orionlibs.spring_http_request_logger.config.MockController#getHome(HttpServletRequest, HttpServletResponse, Model) took 16563800ns
